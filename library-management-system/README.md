# Library Management System

## Description
A Python-based command-line Library Management System designed to manage
books, track quantities, and handle borrow and return operations. This
project demonstrates practical use of Object-Oriented Programming and
file handling in Python.

## Features
- Add, view, search, update, and delete books
- Borrow and return books
- Track original and available quantities
- Save and load data using file handling
- Menu-driven interface

## Concepts Used
- Classes and Objects
- Dictionaries
- File Handling
- Exception Handling
- __str__ method
