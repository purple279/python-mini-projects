# Student Marks Management System

## Description
The Student Marks Management System is a Python-based command-line
application used to manage student academic records. It allows users
to add, view, update, delete, and search student details while storing
data persistently using file handling.

This project is implemented using Object-Oriented Programming principles.

## Features
- Add new student records
- View all students
- Search student by ID
- Update student name or marks
- Delete student records
- Save and load student data from file
- Menu-driven command-line interface

## Concepts Used
- Object-Oriented Programming (Classes & Objects)
- Dictionaries
- File Handling
- Exception Handling
- __str__ method

## Menu Options
1. Add Student  
2. View Students  
3. Search Student  
4. Save to File  
5. Update Student  
6. Delete Student  
7. Exit
